# pyImageColorAnalysisTool
This tool is designed to analysis color distribution of image based on k-means cluster.

# Usage
## Download
```text
git clone https://github.com/Doradx/pyImageColorAnalysisTool.git
```
## Install requirement
```text
pip install requirements
```
### Run
```text
python main.py
```
